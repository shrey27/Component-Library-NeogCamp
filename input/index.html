Input

A class to style any type of input element and handle the error messages.





Add 'ipt' class to style primary input fields. In case mandatory fields or fields with required attributes add 'ipt--man' for error handling. Add dimensions to input-container to limit the width of fields. 

              
              Copied!
              
<label for="user-name">
    <span class="lbl">Username</span>
    <input type="text" placeholder="Enter Username" class="ipt"
           id="user-name" name="user-name" />
</label>

<label for="pswd">
  <span class="lbl">Password*</span>
   <input type="password" placeholder="Enter Password" class="ipt ipt--man"
          id="pswd" name="password" required />
    <span class="ipt--error">This field is mandatory</span>
 </label>
