Getting Started

Apollo-UI is an easy to understand, modular and OOCSS structured vanilla-CSS library. This is a one-stop solution to designing problems. Go through the documentation to learn how to use each and every component. Before launhing your project, let's get started with the installation.

Installation Guide

In order to introduce apollo-ui to your project, Copy the links given below and paste them in the 'head' tag of your html, above all other style sheets. To use the icons download any open-source library or you can check out the git repo for font-awesome library used in this library. 

            <link rel="stylesheet" href="https://apolloui-neog.netlify.app/utility/common.css">

For dynamic behaviour add the below given script tag in body of your document. 

<script src="https://apolloui-neog.netlify.app/utility/script.js"></script>
